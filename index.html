Formulário

Formulário com GET

O método GET passará os dados do formulário como parâmetros da URL, como quando realizamos uma busca. Não utilize o GET para inputs com informações sensíveis como senhas, pois elas aparecerão na URL.

Formulário com POST

O método POST passará os dados do formulário por baixo dos panos. Utilizamos para tratar dados sensíveis como senhas e e-emails
